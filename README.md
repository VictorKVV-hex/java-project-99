### Описание
Task Manager – система управления задачами. Она позволяет ставить задачи, назначать исполнителей и менять их статусы. 
Для работы с системой требуется регистрация и аутентификация.
Проект позволяет создавать новые задачи, удалять и редактировать существующие. 
У задач есть такие аттрибуты, как заголовок, содержание, статус, метка, дата создания и назначенный исполнитель.
